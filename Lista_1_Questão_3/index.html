Qual é a diferença entre linguagem de programação Server Side e Client Side?

A diferença entre a linguagens Server side e Client side São, o Server side ou Back-end como os desenvolvedores chamam é o lado Servidor, o lado servidor tem os bancos de dados ao qual nao é visível ao cliente.
o Client side ou Front-End como os desenvolvedores chamam é o lado cliente, como o nome ja diz cliente é lado que vamos visualizar as paginas web,aplicativos.

Server Side ou Back-end
linguagens utilizadas no Server Side

  • PHP
  • ASP
  • JAVA
  • RUBY
  • PYTHON

Client Side ou Front-end
linguagens utilizadas no Client Side

  • HTML
  • JAVASCRIPT
  • CSS
