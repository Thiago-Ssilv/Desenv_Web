Aplicativos que auxiliam nos estudos de programação

1-MIMO

O Mimo foi o primeiro aplicativo que ouvi falar, usei e achei bastante interessante tem várias linguagens de programação como HTML,JAVASCRIPT,PYTHON.
A forma de ensino é bastante clara através de imagens e opções de múltiplas escolhas nas respostas.

2-PROGRAMMING HUB

O Programming Hub ele é bem parecido com o Mimo porém ele é mais teorico, o Programming Hub explica o conceito da programação e depois passa a falar das linguagens de programação.
O Programming Hub tem as linguagens de java,python,javascript e html.
